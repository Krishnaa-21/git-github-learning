# Git Init & Clone 

## Overview

Before working with Git and GitHub projects, two foundational commands are essential:

- `git init`
- `git clone`

These commands help you start a new Git project or copy an existing one.

Understanding them clearly is critical for every developer.

---

# 1. Git Init

## Definition

`git init` initializes a new Git repository in your current project folder.

It tells Git:

> "Start tracking this project."

After running this command, Git creates a hidden `.git` directory that stores:

- commit history
- branches
- configuration
- staging information
- repository metadata

git init
